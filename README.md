# Verilog Digital Design Projects

I created these Verilog projects as part of my learning journey in digital design and RTL development. Through these implementations, I practiced fundamental hardware design concepts such as combinational and sequential circuits, FSM design, arithmetic units, counters, registers, and digital system modeling.
## Projects Included

| Module                   | Description                                   |
| ------------------------ | --------------------------------------------- |
| Simple Adder             | Binary addition circuit                       |
| D Flip-Flop              | Edge-triggered storage element with set/reset |
| Counter                  | Sequential binary counter                     |
| 4-Bit Shift Register     | Serial data shifting circuit                  |
| Ring Counter             | Circular shift register implementation        |
| Priority Encoder         | Encodes the highest-priority active input     |
| 16:1 Multiplexer         | Selects one of sixteen inputs                 |
| BCD to 7-Segment Decoder | Drives a seven-segment display from BCD input |
| ALU                      | Performs arithmetic and logical operations    |
| Parity Generator         | Generates parity bits for error detection     |

## Concepts Covered

* Combinational Logic Design
* Sequential Logic Design
* Register Transfer Level (RTL) Design
* Finite State Machines (FSM)
* Counters and Registers
* Multiplexers and Encoders
* Arithmetic and Logic Operations
* Error Detection Techniques
* Clocked Digital Systems
